## E45: 'readonly' option is set 

```bash
export PATH={$Home}/.rbenv/bin:$PATH && \
eval "$(rbenv init -)"
```

이 코드 실행 후 `E45: 'readonly' option is set (add ! to override)` 라는 오류가 뜰 수 있다. 
이것은 현재 편집 중인 파일이 읽기 전용이라서 그렇다. 

그냥 `:w!`로 강제 저장해주어 해결했다.
