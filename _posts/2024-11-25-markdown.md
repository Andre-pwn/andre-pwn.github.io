🍀 jekyll 블로그 포스팅을 위한 Markdown 문법 정리

## 제목(Header)
---
글의 제목을 작성할 때 사용하며, 각 제목마다 permalink가 있다.  
`<h1>`,`<h2>`,`<h3>`,`<h4>`,`<h5>`,`<h6>` 태그로 변환되고,
'#'의 갯수에 따라 제목 크기가 나뉜다.
```markdown
# h1
## h2
### h3
#### h4
##### h5
###### h6
```
# h1
## h2
### h3
#### h4
##### h5
###### h6

`<h1>`, `<h2>`는 다음과 같이 표현할 수 있다.
```markdown
h1
=====

h2
-----
```
## 강조(Emphasis)
---
텍스트에 `<em>`(기울임), `<del>`(취소선), `<strong>`(두꺼움), `<u></u>`(밑줄) 등을 표현할 수 있다.

### 두꺼움
---
```markdown
**굵게 쓰기**
```
**굵게 쓰기**

### 기울임
---
```markdown
*기울여 쓰기*
***굵고 기울여 쓰기***
```
*기울여 쓰기*
***굵고 기울여 쓰기***

### 취소선
---
```markdown
~~텍스트 취소하기~~
```
~~텍스트 취소하기~~

### 밑줄
---
```markdown
<u>텍스트에 밑줄</u>
```
<u>텍스트에 밑줄</u>

## 줄바꿈
---
`줄바꿈(Line Breaks)`은 문장 마지막에 띄어쓰기 2번 이상,  
또는 문장 마지막에 `<br>`을 입력하면 된다.
```markdown
안녕하세요.  <!--띄어쓰기 2칸-->
좋은 날이에요.
```
안녕하세요.  
좋은 날이에요.

```markdown
안녕하세요.<br>좋은 날이에요.
```
안녕하세요.<br>좋은 날이에요.
